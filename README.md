# WORLD--ENGINE
A civilizational simulation exploring what happens when you give AI agents cognition, values and collective intelligence- separatly and together

## What This Is
A controlled experiment comparing four configurations:
- Baseline (template behaviour)
- BDI Only (cognitive artitecture)
- BDI + Value (cognition + value system)
- Full AGI (cognition + values + collective intelligence)

## Key Finding 
Pure cognitive architecture (BDI) alone produced 0 behavioural change across all matrics. Value were the activation mechanism that made cognition meaningful.

## How to Run 
python run_experiment.py --ticks 400 --runs 3
